# Mendelow Official Site
[https://mendelow.chatopera.com](https://mendelow.chatopera.com)

## Setup

* Install dev

```
bin/install.sh
```

* Run preview & development

```
bin/dev.sh
```

## Posts

* Create new post

```
bin/create-post.sh your post title
```


## Publish

```
bin/commit.sh # check build status, https://github.com/MendelowApp/MendelowApp.github.io
```
